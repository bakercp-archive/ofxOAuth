Callback Params

  • {@QUERY} is replaced with the full query.
  • {@GET_PARAMS} is replaced with the get query params.
  • {@POST_PARAMS} is replaced with post params.
  • {@HEADERS} is replaced with headers.
  • {@COOKIES} is replaced with cookies.
